How to contribute

  1. Fork this repository using the "Fork" button at the top right of the page.
  2. Edit index.html and add your training under the appropriate topic card above.
    Please follow the format below: 
    <ul>
  <li>Training title or description [with a public link if available]</li>
</ul>
  3. Commit your changes with a clear message.
  4. Open a Pull Request from your forked repository to the main branch of this repository.
  5. Your PR will be reviewed and merged. Thank you for contributing!

Need help? See the GitHub contributing guide.
